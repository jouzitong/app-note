= N5 核心句型
:toc: left
:toclevels: 2
:sectnums:

== 判断句（です）

- A は B です：A 是 B
* 例：私は学生です。= 我是学生。

- A は B ではありません：A 不是 B
* 例：これは本ではありません。= 这不是书。

- A は B ですか：A 是 B 吗
* 例：あなたは先生ですか。= 你是老师吗？

== 指示与存在

- これは / それは / あれは ... です：这是/那是...
* 例：これは辞書です。= 这是词典。

- （人・动物）N が います：有（人/动物）
* 例：教室に先生がいます。= 教室里有老师。

- （事物）N が あります：有（事物）
* 例：机の上に本があります。= 桌子上有书。

== 助词核心用法

- N は：提示主题
* 例：私は日本人です。= 我是日本人。

- N が：提示主语
* 例：猫がいます。= 有猫。

- N を V：宾语
* 例：ご飯を食べます。= 吃饭。

- N に：时间、目的地、对象
* 例：七時に起きます。= 7 点起床。

- N へ：方向
* 例：学校へ行きます。= 去学校。

- N で：动作发生地点、手段
* 例：図書館で勉強します。= 在图书馆学习。

- N と：和、引用
* 例：友達と話します。= 和朋友说话。

- N の N：所属、修饰
* 例：私の本です。= 是我的书。

== 动词礼貌形（ます）

- V ます / V ません / V ました / V ませんでした
* 例：毎日日本語を勉強します。= 每天学习日语。
* 例：今日は行きません。= 今天不去。
* 例：昨日映画を見ました。= 昨天看了电影。

== 时间表达

- （时间）に V：在...做...
* 例：六時に起きます。= 6 点起床。

- 〜から 〜まで：从...到...
* 例：九時から五時まで働きます。= 从 9 点工作到 5 点。

- いつ V ますか：什么时候...
* 例：いつ日本へ行きますか。= 你什么时候去日本？

== 场所与移动

- （地点）に 行きます / 来ます / 帰ります
* 例：日本に行きます。= 去日本。

- （地点）で V：在某地做动作
* 例：駅で友達に会います。= 在车站见朋友。

- （地点）へ どうやって 行きますか：怎么去...
* 例：学校へどうやって行きますか。= 怎么去学校？

== 请求与许可

- V てください：请...
* 例：もう一度言ってください。= 请再说一遍。

- V ても いいです：可以...
* 例：ここで写真を撮ってもいいです。= 可以在这里拍照。

- V ては いけません：不可以...
* 例：ここでタバコを吸ってはいけません。= 这里不能抽烟。

== 进行与习惯

- V ています：正在...；处于某状态
* 例：今ご飯を食べています。= 现在正在吃饭。

- いつも / よく / ときどき + V ます：频率表达
* 例：私はよく音楽を聞きます。= 我经常听音乐。

== 希望与意愿

- N が ほしいです：想要...
* 例：新しいかばんがほしいです。= 想要新包。

- V たいです：想做...
* 例：日本へ行きたいです。= 想去日本。

== 能力与可能

- V ことが できます：会...；能...
* 例：私は日本語を話すことができます。= 我会说日语。

- N が わかります：懂...
* 例：この文の意味がわかります。= 我懂这句话的意思。

== 比较与程度

- A は B より ... です：A 比 B ...
* 例：今日は昨日より寒いです。= 今天比昨天冷。

- A と B と どちらが ... ですか：A 和 B 哪个更...
* 例：日本語と英語とどちらが難しいですか。= 日语和英语哪个更难？

- いちばん ...：最...
* 例：この店がいちばん安いです。= 这家店最便宜。

== 原因与转折

- ... から、...：因为...所以...
* 例：雨ですから、出かけません。= 因为下雨，所以不出门。

- でも / しかし：但是
* 例：この店は高いです。でも、おいしいです。= 这家店贵，但是好吃。

== 连接与并列

- A て、B：A 然后 B（动作连接）
* 例：朝ご飯を食べて、学校へ行きます。= 吃完早饭去学校。

- N と N：和（完整列举）
* 例：りんごとバナナを買います。= 买苹果和香蕉。

- N や N など：...和...等（不完整列举）
* 例：机の上に本やノートなどがあります。= 桌上有书和笔记本等。

== 常用疑问句型

- 何ですか：是什么
* 例：これは何ですか。= 这是什么？

- どこですか：在哪里
* 例：トイレはどこですか。= 厕所在哪里？

- いくらですか：多少钱
* 例：このりんごはいくらですか。= 这个苹果多少钱？

- どうですか：怎么样
* 例：日本の生活はどうですか。= 日本的生活怎么样？

== 学习建议

- 先掌握：判断句 + 助词 + ます形 + て形
- 每个句型至少造 3 句（肯定/否定/疑问）
- 结合 N5 高频词汇一起记忆，效果更好
