== 二、浊音

在清音右上角加「゛」变为浊音。

=== が行（由か行浊化）

|===
|平假名 |片假名 |发音

|が |ガ |ga
|ぎ |ギ |gi
|ぐ |グ |gu
|げ |ゲ |ge
|ご |ゴ |go
|===

== 五、促音（小っ）

小「っ」表示停顿一拍。

例：
がっこう（学校）
きって（邮票）

== 六、长音

平假名：
おかあさん（妈妈）

片假名：
スーパー（超市）

== 七、总结说明

1. 清音共 46 个
2. 浊音由 かさたは 行变成
3. は行可以变成 ば行（浊音）和 ぱ行（半浊音）
4. 拗音必须是 い段 + 小ゃゅょ
5. 促音表示停顿
6. 长音表示拖长发音
