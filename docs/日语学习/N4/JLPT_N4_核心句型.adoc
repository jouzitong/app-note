== 普通形与基础扩展

- 普通形（辞书形 / ない形 / た形）+ と思います：我觉得……
* 例：明日は雨が降ると思います。= 我觉得明天会下雨。

- 普通形 + と言います：说……
* 例：田中さんは明日来ると言いました。= 田中说他明天来。

- 普通形 + でしょう：……吧（推测）
* 例：この問題は難しいでしょう。= 这道题很难吧。

- 普通形 + かもしれません：也许……
* 例：彼はまだ会社にいるかもしれません。= 他也许还在公司。

== 意愿・计划・决定

- V（辞书形）つもりです：打算……
* 例：来年日本へ留学するつもりです。= 我打算明年去日本留学。

- V（辞书形）予定です：计划……
* 例：午後三時に会議を始める予定です。= 计划下午三点开始会议。

- V（辞书形）ことにします：决定……（主观决定）
* 例：毎日日本語の日記を書くことにします。= 我决定每天写日语日记。

- V（辞书形）ことになります：变成……；被决定……
* 例：来月大阪へ転勤することになりました。= 已决定我下个月调去大阪。

== 经验・变化・状态

- V（た形）ことがあります：有过……经验
* 例：富士山に登ったことがあります。= 我爬过富士山。

- V（辞书形）ようになります：变得会……
* 例：日本語の新聞が読めるようになりました。= 我变得能看日语报纸了。

- V（ない形）ようにします：尽量不……；努力做到……
* 例：夜遅くまでスマホを見ないようにしています。= 我尽量不看到太晚。

- V（て形）います：持续状态/习惯
* 例：この会社で十年働いています。= 我在这家公司工作十年了。

== 请求・许可・禁止・建议

- V（て形）もいいですか：可以……吗
* 例：ここに座ってもいいですか。= 可以坐这里吗？

- V（て形）はいけません：不可以……
* 例：この部屋でたばこを吸ってはいけません。= 这个房间不能抽烟。

- V（ない形）ないでください：请不要……
* 例：ここで写真を撮らないでください。= 请不要在这里拍照。

- V（た形）ほうがいいです：最好……
* 例：疲れているなら、早く寝たほうがいいです。= 累了的话最好早点睡。

- V（ない形）ほうがいいです：最好不要……
* 例：薬を飲んだあとで、酒を飲まないほうがいいです。= 吃药后最好别喝酒。

== 动作顺序・准备

- V（て形）から、V：做完……再……
* 例：宿題をしてから、テレビを見ます。= 做完作业再看电视。

- V（て形）おきます：事先做好……
* 例：会議の前に資料をコピーしておきます。= 开会前先把资料复印好。

- V（て形）みます：试着做……
* 例：この料理を作ってみます。= 我试着做这道菜。

- V（て形）しまいます：做完了；不小心……
* 例：財布を電車に忘れてしまいました。= 我不小心把钱包落在电车上了。

== 并行・授受・受益

- V（ます形去ます）ながら、V：一边……一边……
* 例：音楽を聞きながら勉強します。= 一边听音乐一边学习。

- V（て形）あげます：为别人做……
* 例：私は友達に日本語を教えてあげました。= 我教了朋友日语。

- V（て形）もらいます：请/得到别人为我做……
* 例：先生に作文を直してもらいました。= 我请老师帮我改了作文。

- V（て形）くれます：别人为我方做……
* 例：母が駅まで迎えに来てくれました。= 妈妈来车站接我了。

== 原因・理由・目的

- 普通形 / N + ので、…：因为……（语气较柔和）
* 例：熱があるので、今日は休みます。= 因为发烧，今天请假。

- V（辞书形）ために、…：为了……
* 例：日本で働くために、日本語を勉強しています。= 为了在日本工作，我在学日语。

- N のために、…：为了……（名词）
* 例：家族のために毎日頑張っています。= 为了家人每天都在努力。

== 条件表达

- V（た形）ら、…：如果……就……；……之后
* 例：時間があったら、一緒に映画を見ましょう。= 有时间的话一起看电影吧。

- V（ば形）、…：如果……（一般条件）
* 例：薬を飲めば、よくなります。= 吃药的话会好起来。

- V（辞书形）と、…：一……就……（客观必然）
* 例：春になると、桜が咲きます。= 一到春天樱花就开。

== 比较・样态・传闻

- N1 は N2 ほど … ない：N1 不如 N2……
* 例：今年の冬は去年ほど寒くないです。= 今年冬天没有去年那么冷。

- N1 より N2 のほうが …：比起 N1，N2 更……
* 例：電車よりバスのほうが安いです。= 比起电车，公交更便宜。

- V（ます形去ます）そうです：看起来要……（样态）
* 例：このケーキはおいしそうです。= 这个蛋糕看起来很好吃。

- 普通形 + そうです：听说……（传闻）
* 例：天気予報によると、明日は雨だそうです。= 据天气预报说明天会下雨。

== 能力・可能・被动（N4 常见）

- V（可能形）ます / られます：能……
* 例：私は日本語でメールが書けます。= 我能用日语写邮件。

- N が 見えます / 聞こえます：看得见 / 听得见
* 例：ここから海が見えます。= 从这里看得见海。

- V（受身形）られます：被……
* 例：私は先生にほめられました。= 我被老师表扬了。

== 数量・限定・程度

- N しか + 否定：只……（仅）
* 例：財布に千円しかありません。= 钱包里只有1000日元。

- N でも：……之类；即使……
* 例：時間がなければ、メールでもいいです。= 没时间的话，邮件也可以。

- V（辞书形）すぎます：过于……
* 例：このかばんは高すぎます。= 这个包太贵了。

== 常用会话句型（N4 高频）

- 〜ていただけませんか：能请您……吗（礼貌请求）
* 例：この書類を見ていただけませんか。= 能请您看一下这份资料吗？

- お / ご + V（ます形去ます）ください：请……（郑重）
* 例：こちらにお名前をご記入ください。= 请在这里填写姓名。

- 〜ように言ってください：请转告……
* 例：田中さんに明日九時に来るように言ってください。= 请转告田中明天9点来。

== 学习建议

- 先掌握：`つもり / 予定 / ことになる / ことにする / たことがある / たら`。
- 每个句型至少造 3 句：肯定、否定、疑问。
- 将句型与 N4 核心词汇联动记忆，优先练“工作、学校、购物、出行”场景。
