== 学习目标

* 能按主题记忆代词，而不是孤立背单词。
* 能系统区分 ko-so-a-do 系列的用法。
* 能区分普通问法和礼貌问法，例如 `だれ / どなた`。
* 能看见一个词就知道它属于哪一类。
* 每个词至少能说 1 个自己的句子。

== 进度控制规则

* 每天学 1 个主题，词少的主题可和复习一起完成。
* 每个主题按“认读 - 理解 - 例句 - 输出”四步走。
* 每天学习时间控制在 25 到 40 分钟。
* 每学完 1 个主题，完成 3 轮输出：
* 第 1 轮：看中文说日语。
* 第 2 轮：看日语说中文。
* 第 3 轮：脱离文本自己造句。
* 复习节奏：
* 当天晚上复习新内容 1 次。
* 第 2 天开始前复习前一天内容 1 次。
* 每 3 天回顾一次已学主题。
* 学完全部主题后做一次总复习。

== 每日执行模板

1. 预习 5 分钟：先看当天主题和词汇列表。
2. 精学 10 到 15 分钟：逐个读词义、看例句、理解主题内部关系。
3. 输出 10 分钟：遮住例句，自己口头造句。
4. 复习 5 到 10 分钟：回顾前一天和前三天主题。

== 容易混淆的重点

* `これ / それ / あれ`：单独使用，后面不直接接名词。
* `この / その / あの`：后面必须接名词，例如 `この本`。
* `ここ / そこ / あそこ`：表示地点。
* `こちら / そちら / あちら`：更礼貌，也可表示方向或人物。
* `だれ` 比较普通，`どなた` 更礼貌。
* `どうして` 和 `なぜ` 都表示“为什么”，前者口语更常见。
* `いくら` 多问价格，`いくつ` 多问数量或年龄，`どのくらい` 多问程度、时间、距离。
* `こんな / そんな / あんな` 常用于描述“这样的 / 那样的”状态。
