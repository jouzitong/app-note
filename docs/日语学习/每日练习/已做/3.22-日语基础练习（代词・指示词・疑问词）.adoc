= 日语基础练习（代词・指示词・疑问词）

== 一、句子填空（核心）

1. 这是我的书。
（　　　）は（　　　）の本です。
答案: これ　私
2. 那是你的包吗？（靠近对方）
（　　　）は（　　　）のかばんですか。
答案:　それ　あなた
3. 那个人是谁？（远）
（　　　）人は（　　　）ですか。
答案:　あの　誰
4. 这个是什么？
（　　　）は（　　　）ですか。
答案:　これ　なん（何）
5. 这是谁的笔？
（　　　）は（　　　）のペンですか。
答案:　これ　誰
6. 那些人是学生吗？（远）
（　　　）人は学生ですか。
答案:　あの
7. 你的名字是什么？
（　　　）の名前は（　　　）ですか。
答案:　あなた　　なん
8. 那个是老师的电脑。（远）
（　　　）は（　　　）のパソコンです。
答案:　あれ　　先生（せんせい）

== 三、选择题（辨析）

1. （この）本は私のです。
2. （それ）は何ですか。
3. （あの）人は先生です。
4. （これ）はあなたのですか。
5. （それ / その）かばんはあなたのですか。
6. （あれ / あの）は何ですか。


== 四、简单造句（输出）

1. 用「これ / わたし」造句

这是我的书：これは私の本です

2. 用「それ / あなた」造句

那是你的包（かばん）：それわあなたのかばんです。

3. 用「あの / 先生」造句

那是田中先生吗？（您是田中先生吗？ ）：あなたわ田中（たなか）さんですか？

4. 用「だれ」造一个疑问句

那个是谁？：あれは誰ですか？

5. 用「なに」造一个疑问句

这是什么？：これは何ですか？
