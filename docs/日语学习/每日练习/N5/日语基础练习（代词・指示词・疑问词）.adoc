= 日语基础练习（代词・指示词・疑问词）

== 一、句子填空（核心）

1. 这是我的书。
（　　　）は（　　　）の本です。

2. 那是你的包吗？（靠近对方）
（　　　）は（　　　）のかばんですか。

3. 那个人是谁？（远）
（　　　）人は（　　　）ですか。

4. 这个是什么？
（　　　）は（　　　）ですか。

5. 这是谁的笔？
（　　　）は（　　　）のペンですか。

6. 那些人是学生吗？（远）
（　　　）人は学生ですか。

7. 你的名字是什么？
（　　　）の名前は（　　　）ですか。

8. 那个是老师的电脑。（远）
（　　　）は（　　　）のパソコンです。

== 三、选择题（辨析）

1. （これ / この）本は私のです。
2. （それ / その）は何ですか。
3. （あれ / あの）人は先生です。
4. （これ / この）はあなたのですか。
5. （それ / その）かばんはあなたのですか。
6. （あれ / あの）は何ですか。
