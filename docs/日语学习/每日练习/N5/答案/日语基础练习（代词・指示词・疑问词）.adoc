=== 1. 日语 → 中文

1. わたし → 我
2. あなた → 你
3. これ → 这个（东西）
4. それ → 那个（靠近对方）
5. あれ → 那个（远）
6. この → 这个（修饰名词）
7. その → 那个（修饰名词，靠近对方）
8. あの → 那个（修饰名词，远）
9. だれ → 谁
10. なに → 什么
11. わたしたち → 我们
12. あなたたち → 你们
13. この人 → 这个人
14. その人 → 那个人（靠近对方）
15. あの人 → 那个人（远）
16. これら → 这些
17. それら → 那些（靠近对方）
18. あれら → 那些（远）
19. どれ → 哪个（东西）
20. どの → 哪个（修饰名词）
