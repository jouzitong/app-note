= 常见句子

== 基础

=== 初次学习

==== ...です

.例句:
私は中国人です。

.单词解析
* 中国人：（平假名：ちゅうごくじん） -> 中国人

.句式
* `です`: 表示肯定。可以理解为 *是...*。
* `ですか`: 表示疑问。可以理解为 *是...吗*？

.拓展句式
[NOTE]
====
* `でしょう`：表示推测性的疑问。*应该是...吧*？
* `ではおりません`: 表示否定。例如: `私は学生ではおりません`（我不是学生）
====


==== ...ます

.例句
* 私たちは毎日日本語を勉強します（我们每天学习日语）
* みんなで学校へ行きます。（大家一起上学）

.单词解析
. `毎日`（まいにち）-> 每天
. `日本語`（にほんご）-> 日本人（日本语）
. `勉強`（べんきょう） -> 学习

.解释
. `...ます`: 表示礼貌，无实际意义。



==== ..は...が好きです。

.例句
僕はサッカーが好きです。（我喜欢足球）


==== 地点 + へ行きます

.例句
* みんなで学校へ行きます。（大家一起学校）
